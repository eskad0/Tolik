# Основной сценарий:

1. Пользователь выбирает “оплата наличными курьеру”;
2. Система переходит в статус заказа.

### Оплата банковской картой:

1. Пользователь выбирает “оплата банковской картой”;
2. Система открывает поля для заполнения реквизитов карты;
3. Пользователь заполняет реквизиты карты;
4. Система производит оплату;
5. Система переходит в статус заказа.

# Недостаточно средств:

1. Пользователь выбирает “оплата банковской картой”;
2. Система открывает поля для заполнения реквизитов карты;
3. Пользователь заполняет реквизиты карты;
4. Система производит оплату;
5. Система выдает сообщение, что на карте недостаточно средств;
6. Система очищает реквизиты карты;
7. Пользователь заполняет реквизиты другой карты;
8. Система производит оплату;
9. Система переходит в статус заказа.
